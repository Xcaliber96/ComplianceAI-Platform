HIPAA Security Rule - Technical Safeguards - Audit Controls (164.312(b))

Standard: Implement hardware, software, and/or procedural mechanisms that record and examine activity in information systems that contain or use electronic protected health information.

Implementation Specifications:
1. Implement procedures to regularly review records of information system activity, such as audit logs, access reports, and security incident tracking reports.
2. Record and examine activity in information systems that contain or use electronic protected health information (ePHI)
3. Maintain audit controls including:
   - User access logs
   - System activity records
   - Security event logs
   - Failed access attempts
   - Data access and modification logs

Requirements:
- Maintain audit trails for all ePHI access
- Log all user actions on systems containing ePHI
- Implement automated audit log collection
- Retain audit logs for minimum required periods
- Regularly review audit logs for security incidents
- Protect audit log integrity
